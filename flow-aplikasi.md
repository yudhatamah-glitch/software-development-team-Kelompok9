A. PENGUNJUNG (PUBLIC)
1. Membuka website
2. Melihat daftar kamar kosong
3. Melihat:
    - harga
    - ukuran
    - foto kamar
4. Login jika ingin lanjut (admin/penghuni)


B. ADMIN
1. Login
    a. Admin login
    b. Masuk ke dashboard

2. Kelola Kamar
    a. Tambah kamar
    b. Edit:
        - harga
        - ukuran (panjang, lebar, tinggi)
        - foto
    c. Status otomatis:
        - kosong / terisi

3. Kelola Penghuni
    a. Tambah penghuni
    b. Pilih kamar
    c. Sistem otomatis:
        - kamar → terisi

4. Kelola Pembayaran
    a. Generate tagihan bulanan
    b. Melihat status:
        - belum
        - cicil
        - lunas
    c. Melihat total dan sisa pembayaran


C. PENGHUNI
1. Login
2. Melihat:
    - data diri
    - kamar
3. Melihat tagihan
4. Melakukan pembayaran (input nominal manual / dummy)
5. Status berubah:
    - belum → cicil → lunas
